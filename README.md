## Команды GIT
```shell
git clone http://gogs.kase.kz:3000/KASE/kase_mobile_android
```

```shell
git push
git pull
```

```shell
git tag

```

```shell
git commit -a -m "Изменения в readme.txt"

```

```shell
git tag --delete version_001
```
